# IncidentIQ MCP Server

A community [Model Context Protocol](https://modelcontextprotocol.io/) (MCP) server for IncidentIQ,
the K-12 service-management platform. It lets AI assistants interact with a district's IncidentIQ
instance to help with IT tickets, asset management, and support workflows.

## Documentation

- [API Permissions](API_PERMISSIONS.md) — which endpoints work with your API key, and how to test them.
- [Claude Desktop Setup](CLAUDE_DESKTOP_SETUP.md) — connecting the server to Claude Desktop.
- [Community Implementation Guide](COMMUNITY_IMPLEMENTATION_GUIDE.md) — deploying and extending the server.

Source, issues, and releases live on [GitHub](https://github.com/IIQ-Community/mcp-incidentiq).
